## 業務項目

### 主要服務

| 服務 | 說明 |
|------|------|
| **故障分析 (FA)** | 電性量測、非破壞性分析、拆解分析 |
| **可靠度測試 (RA)** | 元件/板級耐久測試、AEC-Q 車規認證、老化測試 |
| **材料分析 (MA)** | 微結構觀察、表面/熱分析、成分鑑定 |
| **FIB 電路修改** | IC 電路除錯（設計前段） |
| **訊號完整性測試** | 高速連接認證、SI 測試 |
| **化學分析** | RoHS/REACH 綠色法規檢測 |
| **MOSFET 後段製程 (BGBM)** | 磨薄、正背面金屬化量產服務 |
| **顧問服務** | ISO 26262 功能安全、ESG 諮詢 |

### 2024–2026 策略平台

- **車用電子：** AEC-Q100–Q200、AEC-Q007 BLR，亞洲唯一 AEC 會員第三方實驗室
- **AI / HPC 驗證：** AI ASIC 可靠度與故障分析，已超過總營收 10%
- **先進封裝：** Sub-7nm、Chiplet、2.5D/3D CoWoS 驗證
- **矽光子 / CPO：** 共封裝光學模組測試（2025 年新平台，與 Lightyear 合作）
- **次世代製程材料 (ALD)：** Sub-2nm 原子層薄膜驗證平台
- **太空電子：** 輻射、極端溫度環境測試
- **DEKRA iST 合資：** LED、PCB/PCBA、車用系統可靠度（歐洲市場）

## 總結評估

宜特是台灣上櫃市場中少見的**半導體驗測服務利基型公司**，30 年技術積累形成難以複製的競爭優勢。業務為半導體複雜度升級的結構性受益者——每次製程節點縮小、封裝架構創新，均帶來增量測試需求。

2025 年獲利下滑屬**主動投資而非需求萎縮**，營收仍維持雙位數成長。2026 年投資主軸為矽光子 / CPO 與 Sub-2nm ALD 平台的商業化進程，若順利放量，EPS 有望創新高。

車用（AEC 認證）提供高毛利穩定收益；AI 晶片驗測為成長引擎；新平台為長期選擇權。主要不確定性在中國競爭壓力與新平台執行時程。
